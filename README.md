# fudebako
A GTK4 (GJS) based desktop shells

## installation

from AUR

```sh
yay -Sy fudebako-git
```

or manually

```sh
git clone --depth=1 https://github.com/mcbeeringi/fudebako
cd fudebako
makepkg -si
cd ..
rm fudebako
```

## usage

```sh
fdbk-menu
```

to show desktop apps
